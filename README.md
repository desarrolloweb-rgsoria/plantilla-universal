# 🏢 Panel de Control - Mi Negocio

**Plataforma profesional de gestión de sitios web para negocios.**

Permite a hoteles, restaurantes, tiendas y servicios crear y administrar su propia página web sin conocimientos técnicos, mediante un panel de control intuitivo y visual.

---

## 🚀 Características Principales

### 📋 Gestión de Contenido
- **Hero**: Fondo con imagen/video (YouTube, Vimeo, MP4), textos personalizables y overlay ajustable.
- **Productos**: Catálogo con nombre, precio, descripción, badge e imagen.
- **Servicios**: Lista de servicios con iconos de Font Awesome personalizables.
- **Nosotros**: Historia con editor de texto enriquecido (Quill.js) e imagen de portada.
- **Galería de Imágenes**: Carrusel 3D estilo Xbox 360 con efecto Coverflow y reflejo.
- **Galería de Videos**: Stack de tarjetas estilo iOS con miniaturas de YouTube.
- **Contacto**: Teléfono, email, dirección, mapa de Google Maps, redes sociales y horario.
- **Configuración**: Nombre, logo, paleta de colores, tipografía y copyright.

### 🎨 Herramientas Visuales
- **Editor de Imagen**: Ajuste de zoom, posición y recorte desde URLs externas.
- **Editor de Texto**: Formato enriquecido con negrita, cursiva, listas y enlaces.
- **Selector de Iconos**: Biblioteca de Font Awesome con +600 iconos categorizados.
- **Selector de Color**: Panel HSV completo, paletas predefinidas (Material, Tailwind, Flat UI) y cuentagotas.

### 🔐 Seguridad y Almacenamiento
- **Autenticación**: Login con credenciales únicas por cliente.
- **Base de Datos**: Supabase (PostgreSQL) para almacenamiento centralizado.
- **Almacenamiento de Medios**: URLs externas (Google Drive, YouTube) sin límites de tamaño.

---

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| **HTML5** | Estructura semántica |
| **CSS3** | Diseño responsive, animaciones, glassmorphism |
| **JavaScript (Vanilla)** | Lógica de negocio, modales, interacciones |
| **Supabase** | Base de datos PostgreSQL + API REST |
| **Quill.js** | Editor de texto enriquecido |
| **Swiper.js** | Carrusel 3D para galería de imágenes |
| **Font Awesome 6** | Biblioteca de iconos |
| **Google Fonts** | Tipografía profesional |
